# Multi-Client Chat Server (Python)

A multi-client communication server built using Python sockets and multithreading.  
This project demonstrates a real-time client-server architecture where multiple users can connect, communicate, and interact concurrently.

---

## 🚀 Features

- Multi-client support using TCP sockets
- Multithreading for handling concurrent connections
- Real-time message broadcasting
- Username-based communication
- Graceful client disconnection handling
- Message logging to file (`chat_log.txt`)
- Timestamped messages

---

## 🧠 Concepts Covered

- Client-Server Architecture
- Socket Programming (TCP/IP)
- Multithreading & Concurrency
- Real-time Communication Systems
- Network-Based Application Design

---

## 🛠 Tech Stack

- Python 3
- Socket Programming
- Threading

---

## 📂 Project Structure
server.py # Handles connections, broadcasting, logging
client.py # Connects to server and sends/receives messages
chat_log.txt # Stores chat history (auto-generated)
